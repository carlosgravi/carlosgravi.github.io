Inteligência Regulatória

254+ normas, 9 reguladores, assistente IA com RAG

Web Scraping Claude AI SQLite
▼ Clique para detalhes

Situação

Necessidade de monitorar e analisar normas regulatórias financeiras brasileiras de múltiplos órgãos (BCB, CMN, CVM, ANBIMA, COAF, ANPD), com centenas de documentos, relacionamentos complexos entre normas e obrigações de compliance dispersas.

Ação

Extractors dedicados com BeautifulSoup para 6 fontes regulatórias + monitoramento do Diário Oficial da União (DOU) via parsing de JSON embarcado em HTML. Banco SQLite com 254+ normas, 75+ obrigações e 78 relacionamentos (altera, revoga, complementa, regulamenta). Dashboard Streamlit 11 páginas: visão geral, filtros por regulador/tema, grafo de relacionamentos com Plotly, calendário regulatório, busca inteligente multi-campo, novidades DOU e assistente IA com RAG (Claude Haiku/Sonnet) para consultas em linguagem natural. Autenticação bcrypt com RBAC e logging em Google Sheets. Pipeline semanal via GitHub Actions.

Resultado

Plataforma completa de inteligência regulatória cobrindo 18 temas (PLD/FT, PIX, LGPD, Fundos, Cybersecurity, etc.) com atualização semanal automatizada e assistente IA para consultas contextuais.

254+ normas 9 reguladores 75+ obrigações 78 relacionamentos 18 temas 11 páginas

RPAs & Automações

30+ automações em pipelines diários e semanais

Python GitHub Actions RPA
▼ Clique para detalhes

Situação

Múltiplos processos manuais e repetitivos em diferentes projetos: coleta de dados de 7 plataformas de mídia, monitoramento de normas regulatórias, consulta a APIs governamentais, geocodificação de empresas, extração de dados de data warehouses e notificações operacionais.

Ação

Desenvolvimento de 30+ RPAs/automações distribuídos em 6 projetos: 7 extractors de mídia digital (Google Ads OAuth2/MCC, Meta 8 contas com cursor pagination e backoff exponencial, TikTok REST v1.3, GA4, Search Console, Business Profile, Orgânico) gerando 57+ CSVs/dia. 6 web scrapers regulatórios com BeautifulSoup (BCB, CMN, CVM, ANBIMA, COAF) + monitoramento do DOU via parsing JSON. 8+ extractors de APIs governamentais (BCB OLINDA, BcBase, SGS, ANTT/CKAN, ANP, BigQuery, CGU, PNCP) com rate limiting adaptativo. Geocodificação em 3 níveis com fallback automático (Nominatim → AwesomeAPI → IBGE) processando 89.511 empresas. Extração Snowflake via PEM key-pair (7 tabelas). Consolidação cross-platform com normalização de métricas e motor de recomendações. Notificações WhatsApp via gateway Baileys self-hosted. Tudo orquestrado via GitHub Actions com continue-on-error, timeouts configuráveis e pipelines diários/semanais.

Resultado

Eliminação de horas de trabalho manual diário. Pipelines 100% automatizados com monitoramento, retry com backoff, cache inteligente e notificações em tempo real. Dados sempre atualizados sem intervenção humana.

30+ automações 57+ CSVs/dia 25+ APIs 89.511 empresas 6 projetos 3 pipelines CI/CD
Python BeautifulSoup REST APIs OAuth2 GitHub Actions Snowflake BigQuery WhatsApp
